YOLOv8 Object Detection on JeVois Pro Camera

Context

As part of my work with the RoboCup club, I contributed to enhancing the visual perception capabilities of our autonomous robots. This project involved implementing the YOLOv8 object detection model on the JeVois Pro smart camera. This implementation included data collection, image labeling, model training, conversion to a compatible format, and deployment on the camera..

Image Labeling

Goal: Build a labeled dataset to train the YOLOv8 model.

  • Captured images using the JeVois Pro camera and a custom Python script
  • Labeled images using DeepLabel (bounding boxes + class IDs)
  • Used GitHub Desktop for version control and collaboration

Model Training

Goal: Train the YOLOv8 object detection model to detect custom classes.

  • Used PyTorch and the Ultralytics package
  • Organized data and defined the structure
  • Trained YOLOv8 using custom parameters: epochs, batch size, input size, and GPU support

Model Conversion

Goal: Convert and deploy the trained model on the JeVois Pro camera.

  • Exported the trained YOLOv8 model to ONNX format
  • Installed and ran the Hailo Software Suite using Docker on Ubuntu
  • Converted the ONNX model to .hef format compatible with JeVois Pro

Conclusion

This project provided end-to-end experience in real-time computer vision, including dataset creation, model training, conversion, and embedded deployment. It strengthened my skills in image labeling, YOLOv8 training, and deploying deep learning models to constrained hardware like JeVois Pro.
